Data Structures

Branch of study that is responsible for studying how data is organized in a computer to atend every different process request and make it more efficient.

Data structures define how all data is accessed, its organization and processing options, sharing and relating available data.

If we have the same data type on one structure, we can call it as "homogeneous". As example, we have "vectors" and "matrices", or, bringing it to the real world, a bookshelf.

Conversely, heterogeneous structures have different data types, such as an agenda or a registration form.

Estruturas de Dados

Ramo de estudo responsável por estudar como dados são organizados em um computador para atender a diferentes requisições de processo e fazer isto de forma mais eficiente.

Estruturas de dados definem como todos os dados são acessados, sua organização e opções de processamento, compartilhando e relacionando os dados disponíveis.

Se nós tivermos o mesmo tipo de dado em uma estrutura, podemos dizer que é uma estrutura homogênea. Como exemplo, temos "vetores" e matrizes. Ou, trazendo para o mundo real, uma prateleira.

Em contrapartida, estruturas heterogêneas possuem diferentes tipos de dados, como uma agenda ou formulário de registro.
