Data Structures

Branch of study that is responsible for studying how data is organized in a computer to atend every different process request and make it more efficient.

Data structures define how all data is accessed, its organization and processing options, sharing and relating available data.

If we have the same data type on one structure, we can call it as "homogeneous". As example, we have "vectors" and "matrices", or, bringing it to the real world, a bookshelf.

Conversely, heterogeneous structures have different data types, such as an agenda or a registration form.
