Arrays

We can think of arrays as a jar or a box for storing some different items (pieces of data) occupying only one place in the shelf (computer memory).

Each programming language has its own way of declaring an array, but they all work as the same way: like a queue that its first index start at 0.

Arrays

Nós podemos pensar em arrays como um jarro ou uma caixa que usamos pra guardar diferentes itens (dados) ocupando apenas um espaço na prateleira (memória).

Cada linguagem de programação possui sua própria maneira de declarar arrays, mas todas elas funcionam da mesma forma: como uma fila, onde seu primeiro lugar começa no 0.
